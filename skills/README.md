## Using these files in Cursor

- **People:** Open from the repo root: `skills/…`
- **Agents:** Point the model at a file path, e.g. `chick/skills/deductions-and-payroll.md`, or add a [Cursor rule](https://docs.cursor.com) that `@`-mentions these docs when working on payroll or UI.
